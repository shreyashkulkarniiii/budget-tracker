**Budget Tracker** is a personal finance app designed for frictionless tracking of daily transactions via UPIs. It directly integrates with your Gmail account to auto-import transactions from GPay, PhonePe, and Paytm, eliminating the need for manual entries.
**Built using** PWA with Expo & React Native (Web). It can be installed directly on iPhone using the browser, giving it a native app feel.
**Key Features**:
- Import transactions directly from your Gmail account using GPay, PhonePe, and Paytm via the app’s direct integration with your email account.
- Manual entries of transactions are supported for those occasions when auto-import fails.
- Spend analysis with support for weekly, category-wise, and monthly trends.
- Transaction history with support for date filters.
- Month-wise breakdown of transactions.
